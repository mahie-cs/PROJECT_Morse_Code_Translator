This is my first official Python project!
It translates text to Morse code, Morse code back to text, and synthesizes pure, high-quality audio on the fly.

---

## FEATURES

* Two-Way Translation: English text to Morse, and Morse back to English.
* Zero Dependencies: Built entirely using Python's native wave and struct modules. No bloatware, no pip install, no nonsense.
* Superior Audio Output: Generates uncompressed, high-fidelity .wav files. 

---

## MORSE TIMING

Standard Morse code uses a strict 1:3 ratio for dots and dashes. I adjusted the dash duration to a `unit_duration * 5` because I am a benevolent developer who cares about accessibility. 

---

## HOW TO RUN

Since you aren't a total casual, you are obviously running this inside a proper Linux terminal. 

1. Clone the repository.
2. Run the script:
```Python
python Project.py
```
3. Choose option 1 or 2.
4. If you chose and ran option 1, look inside your directory for `morse_output.wav`.
